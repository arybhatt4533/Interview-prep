🤖 AI Interview Pro - Elevate Your Preparation
AI Interview Pro ek intelligent platform hai jo candidates ko unke Dream Jobs ke liye taiyar karta hai. Ye sirf resume se text nahi nikalta, balki us data ko analyze karke ek "Real-Life Simulation" create karta hai taaki aap actual interview se pehle apni galtiyan sudhar sakein.

🚀 Ye Interview Mein Kaise Help Karta Hai?
Sirf resume upload karne ke baad ye app aapke liye ye 3 bade kaam karta hai:

Personalized Questioning: Aapke projects (jaise Omni-Eye ya STDAMS) aur technical skills (Python, SQL, Kali Linux) ke base par exact wahi sawal puchta hai jo interviewer puch sakta hai.

Skill Gap Analysis: AI ye identify karta hai ki aapke resume ke hisaab se kaunsa technical area weak hai aur usi par focus karke aapki "Deep Knowledge" test karta hai.

Real-Time Simulation: Ek "Interviewer-Candidate" environment generate karta hai jahan aap bina kisi pressure ke apne answers ko refine kar sakte hain.

🛠️ Key Technical Highlights
Gemini 1.5 Flash Engine: Latest Google AI model jo natural aur human-like conversations handle karta hai.

Context-Aware Logic: Ye app yaad rakhta hai ki pichle sawal ka aapne kya jawab diya, aur usi se follow-up sawal generate karta hai.

Secure & Private: .env integration se aapka data aur API keys ekdam safe rehti hain.

🏗️ How to Get Started
Install: pip install -r requirements.txt

Key Setup: .env file mein apni API key dalein.

Launch: python app.py chalaein aur apna interview session shuru karein!
