# 🤖 AI Interview Pro: Personalized Technical Simulation

**AI Interview Pro** ek smart platform hai jo candidates ko unke actual technical interviews ke liye taiyar karta hai. Ye sirf ek resume parser nahi hai, balki ek **AI-driven Interviewer** hai jo aapke skills aur experience ke basis par aapse real-time technical sawal-jawab karta hai.

---

## 🚀 Ye Project Interview Mein Kaise Help Karta Hai?

Sirf resume upload karne ke baad, ye app aapke prep ko "Targeted" banata hai:

*   **Customized Questioning:** Aapke specific projects (jaise *Omni-Eye* ya *STDAMS*) aur technical stack (Python, SQL, Cybersecurity) ke base par exact wahi sawal puchta hai jo industry experts puchte hain.
*   **Contextual Depth:** Gemini 1.5 Flash ka use karke ye app ek conversational flow banata hai, jahan aapke pichle jawab se naye "follow-up" questions nikalte hain.
*   **Gap Identification:** Interview ke dauran ye aapko aise topics par push karta hai jahan aapka resume "High Skills" claim karta hai, taaki aap apni taiyari check kar sakein.
*   **Fear Reduction:** Ek safe environment jahan aap bina kisi hesitation ke technical terms explain karne ki practice kar sakte hain.

## 🛠️ Core Tech Stack

*   **AI Engine:** Google Gemini 1.5 Flash (Latest `google-genai` SDK).
*   **Backend:** Flask (Python 3.13).
*   **Security:** `python-dotenv` for API Protection.
*   **Parsing:** PDF extraction module for resume analysis.
